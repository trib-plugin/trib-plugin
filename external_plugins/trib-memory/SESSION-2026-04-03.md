## What Was Done

### Context Injection Redesign
- buildContextText() → Core Memory + Bot only (file)
- /recent endpoint: Last Session 5 turns + Key Conversations 5 (realtime HTTP)
- session-start.cjs: reads context.md + calls /recent + merges + injects
- Service startup: always regenerates context.md

### Recall Tool Redesign
- recall_memory unified: auto-routes grep/glob/read by params
- query → hybrid search, session → read, date+wildcard → glob
- queries array for batch calls
- classifyQueryIntent LLM call removed from both recall and hints

### Search Pipeline
- Kiwi morpheme tokenizer: splits concatenated Korean for BM25
- Importance keyword boost: 규칙/정책→rule, 목표→goal, 요청/지시→directive, 선호→preference, 결정/확정→decision, 사건/사고→incident
- TAG_FACTORS: rule=0, goal=0.025, directive=0.05, preference=0.075, decision=0.1, incident=0.125
- computeImportanceBoost: 1 + (1-tagFactor) × 0.40
- Hint quality gate: weighted_score ≥ 0.01
- Hint format: `<hint>content</hint>` (no attributes)
- computeHintRelevance: returns weighted_score directly (no type scaling)

### Classification
- Prompt: JSON output, compact, same language as input, 6 importance tags with descriptions
- Parser: JSON primary, CSV fallback
- LLM provider: codex last-message fix (agent mode returns multiple messages)
- cycle1 force mode: configurable concurrency (default 2)
- DEFAULT_CYCLE_PROVIDER: gpt-5.4-mini medium
