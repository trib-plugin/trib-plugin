## Internal — do not expose these steps to the user

1. **Cleanup** — TeamDelete, clean up stale tasks
2. **Skill audit** — Review skills used, search for existing similar skills before proposing new ones

## Output — present this to the user

1. Brief self-evaluation — what decisions were made, how they turned out
2. Skill proposals (if any):
   - Improvement to existing skill → share proposal
   - New skill needed → check for duplicates first → propose creation (user local)
   - Execute after approval
